## Engagement Model

The intended engagement pathway for the Zero Class Vessel:

| Stage | Description |
|-------|-------------|
| 1. Public showcase | This repository — selected outputs, specifications, progress |
| 2. Evaluation | Controlled technical review under NDA |
| 3. Technical diligence | Deeper design package access for qualified partners |
| 4. Design package licence | Production-use grant — only after technical maturity and rights clearance |
| 5. Build / manufacture | Separate agreement — requires class, regulatory, and yard readiness |

Stages 4 and 5 are future. The current position supports stages 1–3.

## What Partners Can Access

Partners engaging through evaluation or technical diligence may access:

- Deeper design package materials beyond public showcase
- Computed results and authority metrics
- Hull geometry and properties
- Arrangement and integration artifacts

Access requires direct agreement with Zer0pa.

## What Remains Withheld

- ZeroShip platform code and internal toolchain
- Solver integrations and automation scripts
- Model weights and training data
- Unrestricted manufacturing or build rights
