# Contributing

Thanks for helping improve the public Hull 20098 surface.

## Scope

- public documentation improvements
- typo fixes
- broken link fixes
- clearer public proof routing

## Out Of Scope

- requests for private code
- requests for CAD or raw CFD estates
- requests for OEM data or runtime credentials

## Pull Requests

- keep changes atomic
- preserve the README contract
- keep proof anchors repo-local
- keep the public boundary explicit
